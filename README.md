# Ozon Parser (Camoufox)

Скрипт для поиска конкретного товара (SKU) в поисковой выдаче Ozon на глубину до 100 позиций. Использует библиотеку Camoufox для обхода защиты.

## Приготовления

Для работы требуется **Python 3.10+**.

1.  **Установите зависимости:**
    ```bash
    pip install camoufox
    ```

2.  **Установите браузер:**
    Библиотека Camoufox требует скачивания своего ядра (на базе Firefox):
    ```bash
    camoufox fetch
    ```

## Запуск

Просто запустите файл через терминал:
```bash
python main.py
```
Можно также через IDE: Visual Studio, Visual Studio Code или PyCharm.

При найденном среди первых 100 товаров по запросу товара с нужным SKU скрипт возвратит информацию о нем в JSON-формате:

Иначе возвратит `not_found`.

4 примера работы:
1-й пример: товар с SKU, который был изначально в Ozon, но распроданный со временем (книга "Западня. Занесенные снегом" Блейка Крауча), из-за чего функция `finding_something_in_ozon` не найдет его среди первых 100 товаров и возвратит `not_found`.
2-й пример: книга Блейка Крауча "Возвращение", которая пока что не распродана и информация о ней отобразится в JSON-формате:
3-й пример: ноутбук MSI с видеокартой RTX 5050.
4-й пример: Nintendo Switch 2 (один из товаров по данному поиску).
